# ai-service

This directory is a historical placeholder.

The current CodeReviewX runtime does not start or call a separate Python
`ai-service`. Active GitHub ingestion, MiMo review orchestration, issue
normalization, trace persistence, and comment preview publishing live in
`backend-java`.

This folder is retained only as a future extraction point for heavier analysis
work, such as repository context loading, Semgrep execution inside review runs,
RAG, or provider orchestration workers.

Do not treat this directory as part of the current runnable system.
